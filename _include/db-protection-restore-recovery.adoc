[.underline]*调度日志复制剧本*

*配置并启动作业模板。*

. 复制先前创建的作业模板。
+
.. 导航至资源 → 模板。
.. 找到ONTAP/CVO 设置模板，然后在最右侧单击“复制模板”
.. 在复制的模板上单击“编辑模板”，并将名称更改为“还原和恢复手册”。
.. 为模板保留相同的库存、项目和凭证。
.. 选择 ora_recovery.yml 作为要执行的剧本。
.. 变量将保持不变，但需要在变量 dst_cluster_ip 中设置 CVO 集群 IP。
.. 单击“Save”。


+

NOTE: 直到您准备好在远程站点恢复数据库时，才会运行此剧本。
