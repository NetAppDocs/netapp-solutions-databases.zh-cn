TR-4250 解决了设计存储解决方案以使用 Oracle 数据库支持 SAP 业务套件产品的挑战。本文档主要关注使用最新一代 SAP 解决方案的业务和 IT 领导者所面临的常见存储基础设施设计、部署、操作和管理挑战。本文档中的建议是通用的；它们并不特定于 SAP 应用程序或 SAP 实施的规模和范围。  TR-4250 假设读者对NetApp和 SAP 产品的技术和操作有基本的了解。  TR-4250 是基于NetApp、SAP、Oracle 和我们客户的技术人员的互动而制定的。
